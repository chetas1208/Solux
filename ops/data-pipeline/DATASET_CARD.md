# Dataset Card — Solux Site Screening v0.1

## Dataset Summary

Curated geospatial dataset for screening utility-scale solar and floating solar
(floatovoltaic) sites across the contiguous USA and India. Each record is an
H3 hexagonal cell (resolution 7, ~5.16 km edge length) with pre-computed
dimension scores and a GO / INVESTIGATE / KILL decision.

## Intended Uses

- **Primary:** Solux backend API — site ranking and report generation
- **Secondary:** GIS analysis, academic research on solar siting

## Out-of-Scope Uses

- Financial investment decisions (not a substitute for professional feasibility study)
- Legal permitting filings
- Regions outside USA + India (v0.1)

## Dataset Composition

| Field | Value |
|-------|-------|
| Geographic coverage | Contiguous USA, India |
| Spatial resolution | H3 resolution 7 (~5 km cell edge) |
| Temporal coverage | Data sources 2021–2024 |
| Record types | Land + reservoir floating candidates |
| Key features | H3 index, centroid, area, scores (8 dimensions), decision |

## Scoring Model

Weighted average of 8 dimension scores, each 0–100:

```
final_score = 0.25×solar + 0.20×grid + 0.15×buildability +
              0.15×vegetation + 0.10×storage + 0.05×atmosphere +
              0.05×power_loss + 0.05×water
```

**Decision thresholds:** GO ≥ 70, INVESTIGATE 45–69, KILL < 45.

## Data Limitations

- H3 resolution 7 cells (~26 km²) are large; parcel-level permitting requires
  higher-resolution analysis.
- Grid proximity uses centerline distance (not interconnection capacity).
- Floating solar feasibility requires bathymetric and ecological review not
  captured in this dataset.
- Solar resource from PVGIS climatology (TMY); extreme weather not modeled.
- Missing sources (manual download required) reduce confidence scores.
